# Method Validation Report — Reverse-Phase HPLC, NCE-001 API

**Method ID:** M-HPLC-001
**Effective date:** 2026-02-02
**Scope:** NCE-001 API and related substances

## Method summary

- Column: Agilent Poroshell 120 EC-C18, 3.0 × 100 mm, 2.7 μm.
- Mobile phase A: 0.1% formic acid in water.
- Mobile phase B: 0.1% formic acid in acetonitrile.
- Gradient: 5% B → 95% B over 10 min; hold 2 min.
- Flow: 0.8 mL/min; column temperature 40 °C; injection volume 5 μL.
- Detection: DAD at 254 nm (reference 360 nm); MS confirmation ESI+.
