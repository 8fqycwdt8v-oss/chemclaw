# chemclaw

A chemical development autonomous agent.

## Status

Early scaffolding — project initialized on 2026-04-22.

## Getting started

```bash
# clone
git clone git@github.com:<owner>/chemclaw.git
cd chemclaw

# (recommended) create a virtual environment
python -m venv .venv
source .venv/bin/activate

# install dependencies
# pip install -r requirements.txt  # once requirements.txt exists
```

## Configuration

Copy `.env.example` to `.env` and populate any required API keys before running.
